405-请求接口仅支持POST请求

此项目仅提供POST提交接口，接收结果回调接口需要外网服务器地址。

请求实例：

url：POST:http://acm.swust.edu.cn/OnlineJudge/judge.do

Content-Type：application/json

body：
 

{
	"input":["12 12","12 12"],
	"output":["24","24"],
	"timeLimit":1000,
	"memoryLimit":65535,
	"judgeId":1,
	"src":"#include<iostream>\nusing namespace std;\nint main()\n{\n\tint a,b;\n\tcin>>a>>b;\n    int c=a+b;\n    cout<<c;\n\n\treturn 0;\n}\n",
	"callBack":"http://acm.swust.edu.cn/OnlineJudge/result.do?key=111&submitId=12"
}

参数列表（json）：

  • 题目id(题目id为本人数据库指定题目id，指定后测试数据为本人系统提供)

  • 输入样例数组（自定义输入样例，输入id之后忽略）

  • 输出样例数组（自定义输出样例，输入id之后忽略）

  • 时间限制：ms

  • 内存限制：kb

  • 判题语言：语言列表1-14

  • 源代码

  • callBack地址：为put请求方式的地址

判题语言支持：

  • GCC

    • GNU C90

    • GNU C99

    • GNU C11

  • G++

    • GNU C++98

    • GNU C++11

    • GNU C++14

    • GNU C++17

  • Java 1.8

  • python2.7

  • python3.6

  • JavaScript

  • C#

  • Ruby

  • GO

判题采用异步回调更新函数，网站服务器将判题任务和该id提交给判题服务器，判题服务器判题结束自动调用回调函数携带判题结果。
如上述任务会将结果返回到callBack的put接口中，网站服务器只需要指定一个PUT类型回调地址即可，参数submitId为了分辨是哪个任务的回调结果。
为了回调接口安全，防止其他人篡改结果，建议外加一个密钥："callBack" :"http://ip:port/demo/update.do?submitId=111&key=1asdf112asdfasdf", 由于这个接口地址由网站服务器指定之后发送给判题服务器的，所以只有这两个服务器知道密钥，所以其他人不能修改结果。
回调接口样例：https://github.com/1510460325/online-judge/blob/master/producer/src/main/java/cn/wzy/producer/controller/JudgeController.java
返回数据（json）：

  • 全局信息：编译错误信息之类的，判题正常为null

  • 判题信息（按参数组数顺序）：

    • 结果：

    • 时间消耗

    • 空间消耗

    • 运行错误信息：运行正常为null
